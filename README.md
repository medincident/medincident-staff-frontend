# МедИнцидент (личный кабинет сотрудника ЛПУ)

Веб-приложение для медицинских организаций: регистрация и анализ нежелательных событий и заявок на техническое обслуживание.

**Live-демо:** https://patient-medincident.ulbwa.bombomeow.ru/

---

## Возможности

- **Учёт НС и заявок** — регистрация, журналы с фильтрацией, статусы, чат с ответственным, привязка заявок к инцидентам
- **Аналитика** — прогноз нагрузки на 7 дней (нейросеть LSTM на TensorFlow.js + классический Holt), автоматическая детекция точек смены режима, поиск аномалий
- **Администрирование** — управление организационной структурой и справочниками, гибкая ролевая модель с режимом замещения
- **PWA** — офлайн-режим с фоновой синхронизацией, установка как приложение, push-уведомления
- **Мини-приложение** — интеграция в Telegram и МАКС
- **Доступность** — тёмная и светлая темы, чёрно-белый режим, масштабирование шрифта

## Стек

Next.js 16, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Recharts, TensorFlow.js, NextAuth + Zitadel, Prisma, Workbox.

## Запуск локально

```bash
npm install
cp .env.example .env   # заполнить переменные окружения
npm run dev
```

Откроется на `http://localhost:3000`. PWA в dev-режиме отключён.

## Запуск через Docker

```bash
docker compose up --build
```
